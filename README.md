# 🌤️ Aura Weather

A premium, glassmorphic weather dashboard built with Vanilla JavaScript and CSS.

![Aura Weather Preview](https://via.placeholder.com/800x450?text=Aura+Weather+Dashboard)

## ✨ Features
- **Modern UI:** Clean glassmorphism design with backdrop filters.
- **Dynamic Themes:** Backgrounds and icons change automatically based on the weather condition.
- **Real-time Data:** Fetches live weather information using the free `wttr.in` API.
- **Weekly Forecast:** View the upcoming weather for the next few days.
- **Responsive:** Optimized for mobile and desktop screens.
- **Micro-interactions:** Smooth transitions and hover animations.

## 🚀 Tech Stack
- **Structure:** Semantic HTML5
- **Logic:** Vanilla JavaScript (ES6+)
- **Styling:** CSS3 (Variables, Grid, Flexbox)
- **Icons:** [Lucide Icons](https://lucide.dev/)
- **API:** [wttr.in](https://wttr.in/)

## 🛠️ Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/aura-weather.git
   ```
2. Open `index.html` in your browser or run with a local server:
   ```bash
   npm install && npm run dev
   ```
